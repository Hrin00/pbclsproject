案 例 学 习 系 统

This system is developed for teaching and learning in course of Software College. It is a learning system for
transferring, tracking, direction and management of case learning content and process rate. This system
imitate a virtual world, you can take any roles you want to act. These roles may be anyone in a practical
project. Enjoy your learning like playing a game!
